Подробное описание

template<typename TgName, typename JParams, typename JResult, typename I, void(I::*)(std::unique_ptr< typename JParams::target >, std::function< void(std::unique_ptr< typename JResult::target >) >) mem_ptr, typename ... Args>
struct wjrpc::invoke_method< TgName, JParams, JResult, I, mem_ptr, Args >

конструкция для описания обработчика конкретного метода

Параметры шаблона
TgNameИмя метода. Задатеться с помощью JSONRPC_TAG
JParamsJSON-описание параметров запроса
JResultJSON-описание структуры ответа
Iтип целевого класса (интерфейса), которые был задан в wjrpc::target
mem_ptrуказатель на функцию-член класс I
Argsдополнительные параметры обработчика метода

С помощью таких конструкций определяеться списко методов для wjrpc::member_list

Объявления и описания членов структуры находятся в файле:
  • wjrpc/method/mem_fun/invoke_method.hpp
